## Create and attach manage disk in azure

On this demo, we will create a managed disk on azure. On the search bar, type ``disk`` then click on the first service that appears.
<br>
<img width="523" height="331" alt="2026-04-05_10-05" src="https://github.com/user-attachments/assets/427302b5-c8bf-4422-9760-a36aa825e639" />
<br>
<br>

Under ```Resources```, click ```Create```.
<br>
<img width="243" height="146" alt="2026-04-05_10-06" src="https://github.com/user-attachments/assets/61758629-78da-4a43-94a4-182ca745ba98" />
<br>
<br>

We will then name this disk as ```nautilus disk``` and set the disk type as ```Standard LRS``` and the size to ```2```GB. Cick on ```Review + create``` once done to create the disk.
