## Attach Network Interface Card (NIC) to Azure Virtual Machine

On this demo, we will Attach Network Interface Card (NIC) to an existing Virtual Machine.
<br>

First, We need to check if the vm is stopped (deallocated) state. To do that, click the existing virtual machine.

<img width="301" height="280" alt="image" src="https://github.com/user-attachments/assets/c0b799b4-9461-4ca5-9ca4-222f62325cbd" />
<br>

Click **Stop** and wait till it's status change to **stopped (deallocated)**. 

<img width="538" height="108" alt="image" src="https://github.com/user-attachments/assets/821c4291-366d-40c8-b222-77e40387f9a9" />
<br>

<img width="745" height="236" alt="image" src="https://github.com/user-attachments/assets/f7bb6688-0bbd-42b0-8404-b39eaee82a9e" />
<br>

<img width="324" height="390" alt="2026-01-13_14-05" src="https://github.com/user-attachments/assets/44dee888-e518-4be1-b2ba-5bb802b89b05" />
<br>

Next, Navigate to **Networking > Network settings** and select **Attach network interface**.

<img width="578" height="442" alt="2026-01-13_14-08" src="https://github.com/user-attachments/assets/71da3a02-186d-456a-a516-4f98c592975e" />
<br>

We will then choose the existing NIC to attach then click **Ok**.

<img width="531" height="131" alt="image" src="https://github.com/user-attachments/assets/c383d393-0903-441f-9447-9e1ed4f3d9ca" />
<br>

Wait for a moment. We can see that it is now attached.

<img width="119" height="59" alt="image" src="https://github.com/user-attachments/assets/826a49c3-bd23-40b0-9591-e69d9caf2a61" />
<br>

If we click it, we can see that it is attached to the VM.

<img width="1492" height="346" alt="2026-01-13_14-14" src="https://github.com/user-attachments/assets/c9bc2f2b-d161-4cdb-b766-973a79677ada" />
