## Create a Private Azure Blob Storage Container

On this demo, we will create an Azure blob storage container.

For this, we will create a new storage account. We can see that it is available on the main dashboard.

<img width="1106" height="227" alt="2026-04-25_18-28" src="https://github.com/user-attachments/assets/e6ba50df-c849-4640-bbea-6e767b62d0ed" />
<br>
<br>

Once we navigate, we will then select ```create```.

<img width="314" height="277" alt="2026-04-25_18-31" src="https://github.com/user-attachments/assets/56b899d5-5746-41b3-a384-c5b9951c9278" />
<br>
<br>

For the account name, indicate your desired name of for the account. For the storage type, we will select the azure blob storage. Once seetled, click on ```Review + create```.

<img width="809" height="897" alt="2026-04-25_18-34" src="https://github.com/user-attachments/assets/626c72bf-f644-48cd-9f82-d7aad2f5e826" />
<br>
<br>

Once created, go to resources then select the storage account that we've just created.

<img width="860" height="426" alt="2026-04-25_18-38" src="https://github.com/user-attachments/assets/cb109d88-7b9e-4ac0-b9ff-575c3c3bd215" />
<br>
<br>

Go to ```Data Storage``` > ```Containers```.

<img width="287" height="623" alt="2026-04-25_18-40" src="https://github.com/user-attachments/assets/d32d3877-6d0a-4c51-bb74-4b65cb1bdf1f" />
<br>
<br>

Click ```Add containers```.

<img width="422" height="88" alt="2026-04-25_18-42" src="https://github.com/user-attachments/assets/8bbde603-d1ad-416c-86c3-108b20fac090" />
<br>
<br>

For this, you can name it as you desire then click ```Create```.
