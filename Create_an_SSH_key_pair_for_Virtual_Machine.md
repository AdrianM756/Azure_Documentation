## Create an SSH key pair for Virtual Machine

For this demo, we will create an SSH key pair with the following requirements:

* The name of the SSH key pair will be ```devops-kp```.

* The key pair type must be ```RSA``` Format.
<br>

On the <b>Azure services</b> section, click on <b>More services</b>.
<img width="1144" height="164" alt="image" src="https://github.com/user-attachments/assets/dee10fa9-b3e1-448a-b5b1-5b09c533f746" />
<br>

On the <b>>Filter services</b>, type <b>ssh</b> Then select <b>SSH Keys</b> and click on <b>Create</b>
<img width="744" height="397" alt="image" src="https://github.com/user-attachments/assets/256c15cf-18c7-4e15-837d-bdc923cc1098" />
<br>

For this section, we will use the default subscription and resource group. Under the <b> Key pair name</b>, we will name it as ```devops-kp``` and the <b>SSH Key Type</b> in ```RSA``` Format.

<img width="780" height="527" alt="2026-01-03_15-15" src="https://github.com/user-attachments/assets/694ce614-8385-4a2b-8553-9c496d280cd6" />
<br>

Next, click on <b>Review or Create</b> then Download the private key.
