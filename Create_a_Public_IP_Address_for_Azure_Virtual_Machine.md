## Create a Public IP Address for Azure Virtual Machine

On this demo, we will be creating a public ip address for our virtual machine in azure.
<br>

On the search bar, type and click ```Public IP addresses```.

<img width="530" height="280" alt="2026-01-13_10-23" src="https://github.com/user-attachments/assets/7d3240b1-94aa-46ab-afaa-6857985dd2aa" />
<br>

Next, click on ```Create```.

<img width="279" height="187" alt="2026-01-13_10-26" src="https://github.com/user-attachments/assets/cea6052b-9846-4219-857e-e2f5b8b59fa3" />
<br>


We will then name it as ```devops-pip``` and assign it to the ```East US``` region. Leave everything as default. Once done, click **Review + create**.

<img width="778" height="886" alt="2026-01-13_10-28" src="https://github.com/user-attachments/assets/d048850a-2080-48bc-8f29-4619cc3b958d" />
<br>

After the validation, click **Create**.
