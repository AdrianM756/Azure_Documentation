## Attach Public IP to Azure Virtual Machine

On this demo, We will attach a Public IP to an existing virtual machine on Azure.
<br>

On the Azure Dashboard, navigate to ```All resources```.
<br>
<img width="1487" height="864" alt="2026-03-01_13-56" src="https://github.com/user-attachments/assets/a7569cd5-320d-4aaa-8266-bdcf792034aa" />
<br>

Click ```devops-pip```>```Associate```. On the ```Resource type```, select ```Network interface``` and the already created network interface. Click ```OK``` once done.
