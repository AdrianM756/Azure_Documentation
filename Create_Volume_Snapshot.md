## Create Volume Snapshot

For this demo, we create a snapshot of an existing volume named ```xfusion-vol``` located on the ```us-east-1``` region.
<br>

Go to ```EC2``` under ```Elastic Block Store```, select ```snapshots```.

<img width="185" height="462" alt="2026-05-17_20-10" src="https://github.com/user-attachments/assets/60c9e4d9-22e7-4b06-ad26-b8e933f68605" />
<br>
<br>

Click ```Create snapshot```. We will then set a tag named ```xfusion-vol-ss``` then click ```Create snapshot```.
