Attaching Managed Disk to Azure Virtual Machine

On this demo, we will atach an existing disk on a existing virtual machine in azure.
<br>

Click on the existing virtual machine.

<img width="272" height="291" alt="2026-01-13_11-03" src="https://github.com/user-attachments/assets/02d66f6a-ca9c-4041-a59b-3f8dbec5d834" />
<br>

Under **Settings**, select **Disks**.

<img width="302" height="604" alt="2026-01-13_11-06" src="https://github.com/user-attachments/assets/f2ac01b2-8e5f-4051-b28e-59e03d0f4692" />
<br>

Under the **Data disks**, click on **Attach existing disks** then select the existing disk.

<img width="555" height="339" alt="2026-01-13_11-08" src="https://github.com/user-attachments/assets/cccf9399-7290-44ed-8cdc-a0f90ed9c221" />
<br>

Once done, click on **Apply**.

<img width="392" height="508" alt="2026-01-13_11-11" src="https://github.com/user-attachments/assets/24ee4b64-0fb8-49d3-a511-8abc720cdca4" />
<br>

Click the disk to verify. We can see the disk state is set to **Attached** and it is managed by the **datacenter-vm**.
